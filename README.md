# ImgKit

Image Toolbox for Crystal (based on MagicWand binding)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     img_kit:
       github: Hentioe/imgkit.cr
   ```

2. Run `shards install`
